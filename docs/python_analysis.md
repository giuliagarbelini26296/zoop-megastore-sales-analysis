# Python Analysis

## Objective

Perform exploratory data analysis and validate data quality before loading the information into a database.

## Activities Performed

* CSV loading using Pandas
* Missing value verification
* Duplicate record verification
* Revenue calculation
* Monthly revenue analysis
* Revenue analysis by category
* Data visualization with Matplotlib

## Revenue Formula

Revenue = Unit Price × Quantity Sold

## Libraries

* Pandas
* Matplotlib

## Outputs

* Monthly revenue trend
* Revenue by category
* Business insights for dashboard development
